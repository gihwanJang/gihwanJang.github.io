<문제>

정수 n이 주어졌을 때, 각 인덱스 i(0<=i<=n)가 해당 정수 i를 이진수로 변환 변환했을 때 1의 갯수를 저장하는 배열 ans배열을 반환하시오.

<Example1>

Input: n = 2
Output: [0,1,1]

<Example2>

Input: n = 5
Output: [0,1,1,2,1,2]

<설명>

  • 각 비트를 저장할 배열을 n+1의 크기로 선언해줍니다.
  • for문을 통해 0~n까지 인덱스에 접근합니다.
    해당 인덱스의 비트에 끝이 0이면 i/2한 인덱스 값의 마지막 비트에 0을 추가한 것과 같기 때문에 해당 인덱스의 값을 그대로 받아오고 1일 경우 i/2한 인덱스 값의 마지막 비트에 1을 추가한 것과 같기때문에 해당 인덱스의 값에 1을 더한 값을 저장합니다.
  • 배열을 반환해 줍니다.

<Code>

class Solution {
   public int[] countBits(int n) {
        int[] bits = new int[n+1];
        for(int i=0; i<=n; ++i)
        	bits[i]= i%2==0?bits[i/2]:bits[i/2]+1;
        return bits;
    }
}
