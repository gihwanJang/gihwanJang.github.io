자료구조 Ch.02 : 트리

이진 트리(Binary Tree)

트리의 모든 노드가 최대 두 개의 자식 노드를 가지는 트리를 이진 트리라고 합니다.
이때 두 노드는 각각 왼쪽 자식 노드와 오른쪽 자식 노드라고 합니다.

<이진 트리의 속성>

  • 각 레벨 l에 있을 수 있는 최대 노드의 수 : 2^(l-1)
  • 높이가 h인 트리에 있을 수 있는 최대 노드의 수 : 2^(h+1)-1
  • 높이가 h인 트리에 있을 수 있는 최소 노드의 수 : h+1
  • 높이가 h인 트리의 노드 수 n의 범위 : h+1<n<2^(h+1)
  • n개의 노드로 만들 수 있는 트리의 최대 높이 : n-1
  • n개의 노드로 만들 수 있는 트리의 최소 높이 : log2(n)

<이진 트리의 종류>

  • 포화 이진 트리 : 모든 노드의 자식 수가 0또는 2인 경우
  • 완전 이진 트리 : 마지막 레벨을 제외하면 모두 채워져 있고, 마지막 레벨의 단말 노드는 왼쪽에서 부터 채워진 형태로 되어있는 경우
    • 모든 중간노드의 자식 수가 2일 때 단말 노드의 수가 n이면 전체 노드의 수는 N = 2n-1
  • 완벽 이진 트리 : 모든 중간 노드의 자식 수가 2이며, 모든 다말은 같은 레벨에 있는 경우
    • 단말 노드의 수가 n이면 전체 노드의 수는 N=2n-1
    • 완벽 이진 트리는 항상 완전 이진 트리임
    • 완전 이진 트리는 완벽 이진 트리가 아닐 수 있음
  • 균형 이진 트리 : 모든 노드의 왼쪽 부분 트리의 높이와 오른쪽 부분 트리의 높이의 차이가 1 이하인 경우
    • 균형인수 : 오른쪽 부분트리의 높이 - 왼쪽 부분트리의 높이

<이진 트리의 순회>

  • in-order(중위 순회) : 왼쪽 자식노드(L), 내 노드(P), 오른쪽 자식노드(R) 순서로 방문
  • pre-order(전위 순회) : 내 노드(P), 왼쪽 자식노드(L), 오른쪽 자식노드(R) 순서로 방문
  • post-order(후위 순회) : 왼쪽 자식노드(L), 오른쪽 자식노드(R), 내 노드(P) 순서로 방문
  • BFS(너비우선 탐색) : 내 노드(P), 내 노드로부터 깊이 1인 노드들, 내 노드로부터 깊이 2인 노드들, ... , 내 노드로부터 깊이 N인 노드들 (N: 나(트리)의 깊이)

<정리>

이진 트리에 대한 구현은 따로 하지않겠습니다.
