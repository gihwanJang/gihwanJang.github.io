<문제>

정수는 숫자의 각 자릿수가 이전 자릿수보다 하나 더 많은 경우에만 연속 자릿수를 갖습니다.
[low, high] 범위 내에서 연속 숫자를 포함하는 모든 정수의 정렬된 리스트를 반환하시오.

<Example1>

Input: low = 100, high = 300
Output: [123,234]

<Example2>

Input: low = 1000, high = 13000
Output: [1234,2345,3456,4567,5678,6789,12345]

<설명>

  • low의 자릿수(lowNum)를 low에 10 정수나눗셈을 실행하여 산출합니다.
  • 해당 자릿수의 첫째 자리수를 기억할 count를 1로 선언해 줍니다.
  • 반복하여 아래의 과정을 진행해 줍니다.
    • 리스트에 저장할 수(num)를 0으로 선언해 줍니다.
    • 해당 num에 대하여 아래의 과정을 low의 자릿수+count/10만큼 반복해 줍니다.
      • 각 자리의 수를 정해줄 j를 count%10으로 정해줍니다.
        이때 j는 각 반복 과정 마다 +1 연산을 해 줍니다.
      • num에 10곱하기 연산을 해줍니다.
        처음에는 해당 num 값이 0이므로 아무 영향을 받지 않지만 이후에는 현재의 자릿수를 왼쪽으로 밀어주는 역할을 합니다.
      • num에 j더하기 연산을 해줍니다.
        위에서 밀어주는 역할을 했으니 이 과정에서는 +1이된 j의 값을 밀고난 빈 자릿수에 넣어주는 역할을 하게 됩니다.
      • num에 10나머지 연산을 하여 0일경우 해당 반복을 종료하게 됩니다.
        해당 리스트의 정수는 앞의 자릿수보다 뒷 자릿수가 1씩 커야 하므로 마즈막에 올 수 있는 수 중 가장 큰 수는 9입니다.
        이때 9가 오고 한번더 반복을 진행하게 될 시 10 곱하기 연산을 하며 끝에서 2번째 자리는 9 즉 ...90의 형태의 수가 오게 될것이고 j는 10이므로 더하게 되면 끝은 결국 10의 배수의 형태가 되게 됩니다.
        끝은 결코 0이될 수 없으니 10의 나머지 연산이 0이되면 해당 반복은 종료하게 됩니다.
    • 위의 과정을 통해 num이 정해지게 되고 num이 해당 범위 내에 있으면 리스트에 포함 시키고 high의 범위를 벗어나게 되면 반복을 종료합니다.
    • 또한 num의 최대가 123456789이기 때문에 해당 값을 넘어가면 반복이 멈추지 않게 때문에 해당 값이 되면 반복을 종료합니다.
    • count값을 증가 시켜 다음 num은 앞자리가 1이 더 큰 수가 오게합니다.
      이때 증가시킨후 값이 10의 배수인 경우 앞자리가 0이되기 때문에 한번더 +1연산을 해줍니다.
  • 위의 반복이 끝나면 리스트를 반환해 줍니다.

<Coding>

class Solution {
	public List<Integer> sequentialDigits(int low, int high) {
        List<Integer> digits = new LinkedList<>();
        int count=1;
        int lowNum=0;
        for(int i=low; i>0; i/=10)++lowNum;
        while(true) {
        	int num=0;
        	for(int i=lowNum+count/10, j=count%10; i>0; --i,++j) {
        		num*=10;
        		num+=j;
        		if(num%10==0) {
        			num=0;
        			break;
        		}
        	}
        	if(num>=low) {
        		if(num<=high) digits.add(num);
        		else break;
        	}
            if(num==123456789)break;
        	++count;
        	if(count%10==0)++count;
        }
        return digits;
    }
}
